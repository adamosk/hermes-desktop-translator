About Hermes Translator

Hermes Translator

Version 1.1.0
A lightweight desktop application for quick and easy translations powered by DeepL's API.

Features

  • Global hotkey for quick access from anywhere
  • Support for multiple languages via DeepL API
  • Translation history tracking and management
  • Real-time translation as you type
  • Dark mode support for comfortable night use
  • API usage monitoring and statistics
  • System tray integration for minimal footprint

Information
